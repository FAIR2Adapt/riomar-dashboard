# Contributing to RiOMar Dashboard

Thank you for your interest in contributing! This project is part of [FAIR2Adapt](https://fair2adapt-eosc.eu).

## Getting started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/riomar-dashboard.git`
3. Install dependencies: `npm install`
4. Start the dev server: `npm run dev`

## Development workflow

- Create a feature branch: `git checkout -b feature/my-feature`
- Make your changes
- Run linting: `npm run lint`
- Run type checking: `npm run typecheck`
- Commit and push your branch
- Open a Pull Request

## Code style

- TypeScript with Vue 3 Composition API
- Linting enforced via ESLint + Prettier (run `npm run lint` before committing)
- Follow existing patterns in the codebase

## Reporting issues

Please use [GitHub Issues](https://github.com/FAIR2Adapt/riomar-dashboard/issues) to report bugs or request features.
